Заседания Комитета по клиентскому сервису
АО "ТехноБанк"

Дата проведения: 15.05.2024
Время: 11:00 - 13:00
Место проведения: Конференц-зал №3, 11 этаж

ПРИСУТСТВУЮЩИЕ:
1. Петрова О.В. - Председатель Комитета, Начальник департамента клиентского сервиса
2. Волков С.А. - Член Комитета, Начальник службы поддержки
3. Сидорова М.В. - Член Комитета, Начальник службы качества
4. Козлов Д.С. - Член Комитета, Главный аудитор

ПОВЕСТКА ДНЯ:
1. Анализ показателей клиентского сервиса за Q1 2024
2. Обсуждение планов по улучшению качества обслуживания
3. Рассмотрение новых каналов обслуживания
4. Планирование обучения персонала

РЕШЕНИЯ:

1. ПО ПЕРВОМУ ВОПРОСУ:
Анализ показателей клиентского сервиса за Q1 2024:

Ключевые показатели:
- Удовлетворенность клиентов: 88% (цель: 95%)
- Время обслуживания: 15 минут (цель: 10 минут)
- Количество жалоб: 50 в месяц (цель: 30)
- Повторные обращения: 25% (цель: 15%)

Каналы обслуживания:
- Телефонная поддержка: 40% обращений
- Онлайн-чат: 30% обращений
- Email: 20% обращений
- Личный прием: 10% обращений

2. ПО ВТОРОМУ ВОПРОСУ:
Утверждены планы по улучшению качества обслуживания:
- Внедрение ИИ-ассистента
- Улучшение мобильного приложения
- Создание единого портала
- Персонализация услуг

3. ПО ТРЕТЬЕМУ ВОПРОСУ:
Рассмотрены новые каналы обслуживания:
- Видеозвонки с консультантами
- Голосовой помощник
- Чат-бот в мессенджерах
- Виртуальная реальность

4. ПО ЧЕТВЕРТОМУ ВОПРОСУ:
Планирование обучения персонала:
- Базовое обучение: 100 человек
- Специализированное обучение: 50 человек
- Сертификация: 25 человек
- Бюджет: 5 млн руб.

ДОПОЛНИТЕЛЬНЫЕ РЕШЕНИЯ:
1. Создать центр обучения клиентскому сервису
2. Внедрить систему мотивации персонала
3. Разработать стандарты обслуживания
4. Создать систему обратной связи
