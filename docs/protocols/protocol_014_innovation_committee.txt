1. Козлова Е.С. - Председатель Комитета, Директор по инновациям

ПОВЕСТКА ДНЯ:
1. Обзор инновационных проектов за Q1 2024
2. Обсуждение стратегии инновационного развития
3. Рассмотрение новых технологических трендов
4. Планирование инвестиций в инновации

РЕШЕНИЯ:

1. ПО ПЕРВОМУ ВОПРОСУ:
Обзор инновационных проектов за Q1 2024:

Завершенные проекты:
- ИИ-скоринг клиентов
- Чат-бот для поддержки
- Мобильное приложение
- Система аналитики

Текущие проекты:
- Блокчейн-платежи (прогресс: 60%)
- Цифровые сертификаты (прогресс: 40%)
- ИИ-ассистент (прогресс: 30%)
- Система рекомендаций (прогресс: 20%)

2. ПО ВТОРОМУ ВОПРОСУ:
Утверждена стратегия инновационного развития:
- Фокус на ИИ-технологиях
- Развитие блокчейн-решений
- Создание экосистемы партнеров
- Международное сотрудничество

3. ПО ТРЕТЬЕМУ ВОПРОСУ:
Рассмотрены новые технологические тренды:
- Квантовые вычисления
- Edge computing
- 5G технологии
- Интернет вещей (IoT)
- Дополненная реальность (AR)

4. ПО ЧЕТВЕРТОМУ ВОПРОСУ:
Планирование инвестиций в инновации:
- Общий бюджет: 100 млн руб.
- ИИ-технологии: 40 млн руб.
- Блокчейн: 25 млн руб.
- Облачные решения: 20 млн руб.
- Прочие инновации: 15 млн руб.

ДОПОЛНИТЕЛЬНЫЕ РЕШЕНИЯ:
1. Создать лабораторию инноваций
2. Развивать партнерства с университетами
3. Создать фонд инноваций
4. Внедрить систему управления инновациями
