Заседания Комитета по кибербезопасности

Время: 10:00 - 12:00
Место проведения: Конференц-зал №3, 13 этаж

1. Смирнов И.А. - Председатель Комитета, Начальник службы ИБ

ПОВЕСТКА ДНЯ:
1. Анализ угроз кибербезопасности за Q1 2024
2. Обсуждение планов по усилению защиты
3. Рассмотрение новых технологий безопасности
4. Планирование обучения персонала

РЕШЕНИЯ:

1. ПО ПЕРВОМУ ВОПРОСУ:
Анализ угроз кибербезопасности за Q1 2024:

Выявленные угрозы:
- Фишинг: 15 атак
- Malware: 8 атак
- DDoS: 3 атаки
- Социальная инженерия: 5 атак

Уровень защиты:
- Общий уровень: 95%
- Критические системы: 98%
- Пользовательские системы: 90%
- Мобильные устройства: 85%

2. ПО ВТОРОМУ ВОПРОСУ:
Утверждены планы по усилению защиты:
- Внедрение ИИ для обнаружения угроз
- Улучшение системы мониторинга
- Внедрение zero-trust архитектуры
- Создание SOC (Security Operations Center)

3. ПО ТРЕТЬЕМУ ВОПРОСУ:
Рассмотрены новые технологии безопасности:
- Квантовая криптография
- Биометрическая аутентификация
- Поведенческая аналитика
- Автоматическое реагирование

4. ПО ЧЕТВЕРТОМУ ВОПРОСУ:
Планирование обучения персонала:
- Базовое обучение: 200 человек
- Специализированное обучение: 50 человек
- Сертификация: 25 человек
- Бюджет: 10 млн руб.

ДОПОЛНИТЕЛЬНЫЕ РЕШЕНИЯ:
1. Создать центр кибербезопасности
2. Внедрить систему управления инцидентами
3. Разработать планы реагирования
4. Создать систему мониторинга угроз
