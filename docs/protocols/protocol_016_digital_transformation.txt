ПРОТОКОЛ № 016
Заседания Комитета по цифровой трансформации
АО "ТехноБанк"

Дата проведения: 26.06.2024
Время: 15:00 - 17:30
Место проведения: Конференц-зал №1, 16 этаж

ПРИСУТСТВУЮЩИЕ:
1. Сидоров А.В. - Председатель Комитета, Руководитель IT-департамента
2. Петров М.И. - Член Комитета, Технический директор
3. Козлова Е.С. - Член Комитета, Директор по инновациям
4. Волков Д.С. - Член Комитета, Начальник службы разработки
5. Смирнов И.А. - Секретарь Комитета
6. Петрова О.В. - Начальник службы ИБ (приглашенный)

ПОВЕСТКА ДНЯ:
1. Обзор прогресса цифровой трансформации за Q1 2024
2. Обсуждение стратегии цифрового развития
3. Рассмотрение новых цифровых технологий
4. Планирование инвестиций в цифровизацию

РЕШЕНИЯ:

1. ПО ПЕРВОМУ ВОПРОСУ:
Обзор прогресса цифровой трансформации за Q1 2024:

Завершенные инициативы:
- Модернизация платежной системы
- Внедрение ИИ-скоринга
- Создание мобильного приложения
- Улучшение веб-банка

Текущие проекты:
- Миграция в облако (прогресс: 70%)
- Внедрение блокчейн-технологий (прогресс: 50%)
- Создание ИИ-платформы (прогресс: 40%)
- Развитие API-экосистемы (прогресс: 60%)

2. ПО ВТОРОМУ ВОПРОСУ:
Утверждена стратегия цифрового развития:
- Создание цифровой экосистемы
- Внедрение ИИ-технологий
- Развитие облачных решений
- Создание инновационных продуктов

3. ПО ТРЕТЬЕМУ ВОПРОСУ:
Рассмотрены новые цифровые технологии:
- Квантовые вычисления
- Edge computing
- 5G технологии
- Интернет вещей (IoT)
- Дополненная реальность (AR)

4. ПО ЧЕТВЕРТОМУ ВОПРОСУ:
Планирование инвестиций в цифровизацию:
- Общий бюджет: 300 млн руб.
- ИИ-технологии: 100 млн руб.
- Облачные решения: 80 млн руб.
- Блокчейн: 60 млн руб.
- Прочие технологии: 60 млн руб.

ДОПОЛНИТЕЛЬНЫЕ РЕШЕНИЯ:
1. Создать центр цифровой трансформации
2. Развивать партнерства с технологическими компаниями
3. Создать фонд цифровых инноваций
4. Внедрить систему управления цифровыми проектами
