Заседания Комитета по управлению активами
АО "ТехноБанк"

Время: 10:00 - 12:00
Место проведения: Конференц-зал №2, 12 этаж

1. Волков А.С. - Председатель Комитета, Начальник департамента активов
2. Петрова Н.В. - Член Комитета, Главный аналитик

ПОВЕСТКА ДНЯ:
1. Анализ портфеля активов за Q1 2024
2. Обсуждение стратегии управления активами
3. Рассмотрение новых инвестиционных возможностей
4. Планирование мероприятий по оптимизации портфеля

РЕШЕНИЯ:

1. ПО ПЕРВОМУ ВОПРОСУ:
Анализ портфеля активов за Q1 2024:

Общая стоимость портфеля: 15 млрд руб.
Структура по типам активов:
- Кредиты: 12 млрд руб. (80%)
- Ценные бумаги: 2 млрд руб. (13%)
- Денежные средства: 1 млрд руб. (7%)

Доходность портфеля:
- Общая доходность: 8.5% годовых
- Кредитный портфель: 12% годовых
- Ценные бумаги: 6% годовых
- Денежные средства: 2% годовых

2. ПО ВТОРОМУ ВОПРОСУ:
Утверждена стратегия управления активами:
- Диверсификация портфеля
- Управление рисками
- Оптимизация доходности
- Соответствие требованиям регуляторов

3. ПО ТРЕТЬЕМУ ВОПРОСУ:
Рассмотрены новые инвестиционные возможности:
- Корпоративные облигации: 500 млн руб.
- Государственные облигации: 300 млн руб.
- Акции голубых фишек: 200 млн руб.
- Альтернативные инвестиции: 100 млн руб.

4. ПО ЧЕТВЕРТОМУ ВОПРОСУ:
Планирование мероприятий по оптимизации портфеля:
- Ребалансировка портфеля
- Управление ликвидностью
- Хеджирование рисков
- Мониторинг производительности

ДОПОЛНИТЕЛЬНЫЕ РЕШЕНИЯ:
1. Создать рабочую группу по оптимизации портфеля
2. Провести стресс-тестирование портфеля
3. Разработать план управления ликвидностью
4. Внедрить систему мониторинга в реальном времени
