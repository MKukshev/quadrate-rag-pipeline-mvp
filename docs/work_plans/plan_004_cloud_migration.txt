ПЛАН РАБОТ
по проекту "Миграция в облако"
АО "ТехноБанк"

Период реализации: 01.06.2024 - 31.03.2025
Бюджет проекта: 35 000 000 руб.
Руководитель проекта: Петров М.И.

ЦЕЛЬ ПРОЕКТА:
Поэтапная миграция IT-инфраструктуры банка в облако для повышения масштабируемости, надежности и снижения операционных расходов.

ЗАДАЧИ ПРОЕКТА:
1. Анализ текущей инфраструктуры
2. Выбор облачного провайдера
3. Создание гибридной архитектуры
4. Миграция некритичных систем
5. Миграция критичных систем
6. Оптимизация облачных ресурсов

ЭТАПЫ РЕАЛИЗАЦИИ:

ЭТАП 1: ПОДГОТОВИТЕЛЬНЫЙ (01.06.2024 - 31.07.2024)
- Аудит текущей инфраструктуры
- Анализ требований к облаку
- Выбор облачного провайдера
- Создание архитектуры гибридного облака
- Планирование миграции
Бюджет этапа: 3 000 000 руб.

ЭТАП 2: СОЗДАНИЕ ОБЛАЧНОЙ ИНФРАСТРУКТУРЫ (01.08.2024 - 30.09.2024)
- Настройка облачной платформы
- Создание виртуальных сетей
- Настройка системы безопасности
- Создание системы мониторинга
- Настройка резервного копирования
Бюджет этапа: 8 000 000 руб.

ЭТАП 3: МИГРАЦИЯ НЕКРИТИЧНЫХ СИСТЕМ (01.10.2024 - 30.11.2024)
- Миграция тестовых сред
- Миграция систем разработки
- Миграция систем аналитики
- Миграция систем отчетности
- Тестирование и оптимизация
Бюджет этапа: 6 000 000 руб.

ЭТАП 4: МИГРАЦИЯ КРИТИЧНЫХ СИСТЕМ (01.12.2024 - 28.02.2025)
- Миграция банковских систем
- Миграция систем безопасности
- Миграция систем мониторинга
- Настройка высокой доступности
- Тестирование производительности
Бюджет этапа: 15 000 000 руб.

ЭТАП 5: ОПТИМИЗАЦИЯ И ЗАВЕРШЕНИЕ (01.03.2025 - 31.03.2025)
- Оптимизация облачных ресурсов
- Настройка автоматического масштабирования
- Финальное тестирование
- Документирование архитектуры
- Обучение персонала
Бюджет этапа: 3 000 000 руб.

ТЕХНОЛОГИЧЕСКИЙ СТЕК:
- Облачная платформа: Microsoft Azure
- Контейнеризация: Docker, Kubernetes
- База данных: Azure SQL Database, PostgreSQL
- Хранилище: Azure Blob Storage
- Сеть: Azure Virtual Network
- Безопасность: Azure Security Center
