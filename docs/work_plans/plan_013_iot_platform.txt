ПЛАН РАБОТ
по проекту "Создание IoT-платформы"
АО "ТехноБанк"

Период реализации: 01.11.2024 - 31.07.2025
Бюджет проекта: 18 000 000 руб.
Руководитель проекта: Волков Д.С.

ЦЕЛЬ ПРОЕКТА:
Создание IoT-платформы для мониторинга активов, автоматизации процессов и создания новых банковских сервисов на основе данных с устройств.

ЗАДАЧИ ПРОЕКТА:
1. Разработка IoT-платформы
2. Создание датчиков и устройств
3. Интеграция с банковскими системами
4. Разработка аналитических решений
5. Внедрение и масштабирование

ЭТАПЫ РЕАЛИЗАЦИИ:

ЭТАП 1: ПОДГОТОВИТЕЛЬНЫЙ (01.11.2024 - 30.11.2024)
- Анализ требований
- Выбор IoT-платформы
- Создание архитектуры
- Формирование команды
Бюджет этапа: 2 000 000 руб.

ЭТАП 2: РАЗРАБОТКА ПЛАТФОРМЫ (01.12.2024 - 28.02.2025)
- Создание IoT-платформы
- Разработка API для устройств
- Создание системы управления
- Интеграция с облачными сервисами
Бюджет этапа: 8 000 000 руб.

ЭТАП 3: СОЗДАНИЕ УСТРОЙСТВ (01.03.2025 - 31.05.2025)
- Разработка датчиков
- Создание устройств мониторинга
- Интеграция с платформой
- Тестирование устройств
Бюджет этапа: 5 000 000 руб.

ЭТАП 4: ВНЕДРЕНИЕ И АНАЛИТИКА (01.06.2025 - 31.07.2025)
- Развертывание в промышленную эксплуатацию
- Создание аналитических решений
- Оптимизация процессов
- Планирование масштабирования
Бюджет этапа: 3 000 000 руб.

ТЕХНОЛОГИЧЕСКИЙ СТЕК:
- IoT-платформа: AWS IoT, Azure IoT Hub
- Протоколы: MQTT, CoAP, HTTP
- Языки: Python, C++, JavaScript
- Аналитика: Apache Kafka, Apache Spark
- Визуализация: Grafana, Kibana

КОМАНДА ПРОЕКТА:
- Руководитель проекта: Волков Д.С.
- IoT-разработчики: 4 человека
- Аналитики данных: 3 человека
- DevOps-инженеры: 2 человека
- Тестировщики: 2 человека

- Камеры видеонаблюдения

ПРИМЕНЕНИЯ:

- Мониторинг оборудования

- Автоматизация процессов
