5. API СПЕЦИФИКАЦИЯ
═══════════════════════════════════════════════════════════════════════════════

5.1 Базовый URL
https://ai-governance.technobank.ru/api/v1

5.2 Аутентификация
Используется JWT токены для аутентификации.

5.3 Основные эндпоинты

5.3.1 Управление моделями
POST /models - Создание новой модели
GET /models - Получение списка моделей
GET /models/{id} - Получение информации о модели
PUT /models/{id} - Обновление модели
DELETE /models/{id} - Удаление модели

5.3.2 Развертывание моделей
POST /models/{id}/deploy - Развертывание модели
POST /models/{id}/undeploy - Остановка модели
GET /models/{id}/status - Статус модели

5.3.3 Мониторинг
GET /models/{id}/metrics - Метрики модели
GET /models/{id}/logs - Логи модели
GET /alerts - Список алертов

5.3.4 Аудит
GET /audit/logs - Логи аудита
GET /audit/reports - Отчеты аудита

6. СХЕМА БАЗЫ ДАННЫХ
═══════════════════════════════════════════════════════════════════════════════

6.1 Основные таблицы

6.1.1 Таблица models
- id (UUID, PRIMARY KEY)
- name (VARCHAR(255), NOT NULL)
- version (VARCHAR(50), NOT NULL)
- description (TEXT)
- developer_id (UUID, FOREIGN KEY)
- status (ENUM: development, testing, production)
- created_at (TIMESTAMP)
- updated_at (TIMESTAMP)

6.1.2 Таблица model_versions
- id (UUID, PRIMARY KEY)
- model_id (UUID, FOREIGN KEY)
- version (VARCHAR(50), NOT NULL)
- file_path (VARCHAR(500))
- checksum (VARCHAR(64))
- created_at (TIMESTAMP)

6.1.3 Таблица deployments
- id (UUID, PRIMARY KEY)
- model_id (UUID, FOREIGN KEY)
- environment (ENUM: test, production)
- status (ENUM: pending, running, stopped, failed)
- deployed_at (TIMESTAMP)
- deployed_by (UUID, FOREIGN KEY)

6.1.4 Таблица audit_logs
- id (UUID, PRIMARY KEY)
- user_id (UUID, FOREIGN KEY)
- action (VARCHAR(100))
- resource_type (VARCHAR(50))
- resource_id (UUID)
- details (JSON)
- timestamp (TIMESTAMP)

13. ЗАКЛЮЧЕНИЕ
═══════════════════════════════════════════════════════════════════════════════
